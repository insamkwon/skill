## 스킬 소개


### 🎯 이 스킬은 뭘까?


Claude Code 대화 세션 전체를 분석하여 6가지 핵심 질문에 답합니다:


┌─────────────────────────────────────────────────────────────┐
│  1. 나는 누구인가?            (Identity)                     │
│     → Primary Role, 전문성 분야, 활동 도메인                │
│                                                             │
│  2. 무엇을 많이 하는가?        (Activity Patterns)          │
│     → 작업 의도 기반 시간 분배, 반복 패턴                   │
│                                                             │
│  3. 언제 하는가?              (Temporal Patterns)          │
│     → 업무 시간 패턴, 집중도, 작업 리듬                     │
│                                                             │
│  4. 무엇을 얻었는가?          (Outcomes & Impact)           │
│     → 실제 완료한 작업, 산출물, 영향도                      │
│                                                             │
│  5. 다음은 무엇인가?          (Next Actions)                │
│     → 미완료 작업, TODO, 추천 포커스                        │
│                                                             │
│  6. 어떻게 더 잘할까?          (Improvement)                │
│     → 비효율 탐지, 개선 제안, Quick Wins                    │
└─────────────────────────────────────────────────────────────┘

## 발동 방법


### 🗣️ 그냥 말하세요


말하는 방법실행되는 것 "세션 요약해줘"전체 세션 분석 "세션 저장"전체 세션 분석 + 파일 저장 "이 대화 요약"전체 세션 분석 "오늘 한 일 정리해줘"오늘 세션만 분석 "이번주 업무 분석해줘"이번주 세션만 분석 "업무 패턴 분석해줘"전체 세션 분석 + 패턴 분석

### ⌨️ 슬래시 명령어


/session-summary                    # 전체 세션 분석
/session-summary --today            # 오늘 세션만
/session-summary --week             # 이번주 세션만
/session-summary --date 2026-03-10  # 특정 날짜
/session-summary --date 2026-03-01~2026-03-07  # 기간 지정
